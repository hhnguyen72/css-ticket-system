# User Stories

## US-001 (Need to change)
As a **support agent**, I want incoming customer emails to be automatically parsed and converted into tickets with a priority level, so that I can focus on resolving issues instead of triaging my inbox.

## US-002 (Need to Change)
As a **support team lead**, I want to view real-time metrics on open ticket volume, average resolution time, and agent workload, so that I can reassign tickets before SLA deadlines are breached.

## US-003 (Need to Change)
As a **customer**, I want to receive an automatic acknowledgement with a ticket ID immediately after submitting a support request, so that I know my issue has been received and can track its progress.
